# 🎨 ElectroAI | Frontend

The visual heart of the ElectroAI ecosystem. This module provides a high-performance, interactive canvas for electrical design.

## ✨ Features & Technologies
- **React.js**: Modular component-based architecture.
- **Tailwind CSS**: Modern, utility-first styling with custom glassmorphism effects.
- **HTML-to-Image**: Capture and download your circuit designs as high-quality PNGs.
- **Lucide & FontAwesome**: Professional iconography.
- **Responsive Design**: Custom views for both Desktop (Sidebar-based) and Mobile (Bottom-sheet/Top-bar based).

## 🛠️ Key Components
- **Canva**: The custom-built drawing engine using JS Canvas/SVG logic.
- **AIChat**: A sleek floating interface to interact with the Gemini AI.
- **Header/TopBar**: Context-aware navigation with project management tools.

## 🚀 Development
To start the frontend:
```bash
npm install
npm run dev
```

## ⚙️ Environment Variables
Make sure to create a `.env` file with:
- `VITE_API_URL`: Your backend URL.
- `VITE_DEVELOPER_PHONE`: For the WhatsApp contact button.
